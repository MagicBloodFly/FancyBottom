# FancyTips
这是一个基于PocketMine-MP的插件,且已经支持PMMP`3.0`API

运行此插件可以使你的服务器拥有自定义的计分板和底部显示

> 安装必须注意的事项

**必须搭配Fancy `CKVIPs`和`EconomyAPI`**
